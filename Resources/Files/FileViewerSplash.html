FileViewer

Intended as a basic pdf file viewer but now able to view several other file types.

Features:

  • Create/Rename/Rearrange/Delete Bookmarks
  • Folder navigation
  • External folder navigation via drag & drop
  • Open supported files in system default applications
  • File renaming and deletion
  • Folder renaming only

Usage:

Creating Bookmarks:

Create bookmarks by dropping files or folders from the file browser pane on to the bookmarks pane.

Renaming Bookmarks:

Double click a bookmark to enter a new title.

Rearranging Bookmarks:

Drag & Drop a bookmark wherever you like to create any arrangement you prefer.

Deleting Bookmarks:

Select the desired bookmark and press delete.

External Folder Navigation:

To instantly navigate to a location, drag and drop a file or folder from Windows File Explorer on to the preview pane.

Open Files and Folders:

Open supported files in system default applications by pressing the spacebar while the file is selected.

Open folders in Windows File Explorer by selecting the folder in the browser pane and pressing the spacebar.

Right clicking and selecting Open with Acrobat opens supported files in Adobe Acrobat if installed.

Renaming and Deletion:

Right click a file or folder and select desired operation.

When viewing files, press CTRL+h to show this page, again to hide.
